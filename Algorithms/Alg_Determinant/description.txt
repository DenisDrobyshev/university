Данные алгоритмы ищут определитель квадратной матрицы.

В файле Jordana есть несколько функций которые при взаимодействии получают два алгоритма (алгоритм Жордана
и алгоритм Миноров и Алг. дополнений), которые находят
определитель квадратной матрицы и сравнивают время работы обоих алгоритмов в миллисек.

Файл input.txt содержит матрицу.

Файл output.txt содержит результаты вычислений.
